  # Install Dependencies
  pip install -r backend/requirements.txt

  # Start Backend
  cd backend && python main.py

  # Start Frontend
  cd frontend && python -m http.server 3000

  # Check Server Status
  curl http://localhost:8001/

  # View API Documentation
  http://localhost:8001/docs

  # Deactivate Virtual Environment
  deactivate

🔧 TROUBLESHOOTING QUICK REFERENCE

  PROBLEM                    │ SOLUTION
  ───────────────────────────┼────────────────────────────────
  Port 8001 in use           │ Change in backend/main.py
  Can't activate venv        │ Run Set-ExecutionPolicy command
  Packages won't install     │ Update pip: pip install --upgrade pip
  Frontend won't connect     │ Check ports, verify backend running
  Image upload fails         │ Check file type, see browser console
  
  → See SETUP_GUIDE.md for detailed troubleshooting
